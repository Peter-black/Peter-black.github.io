RAILS TO REST (#RAINBOWJAM16) - SEPTEMBER 2016

(Skip to 9:12 in the video for a playthrough) As a team we decided to enter the #RainbowJam16 which had a 16 day period to make a game with the theme 'Identity'. We were very busy and so we were only able to use the final 5 days on the jam.

We used the game to experiment with new things that we hadn't tried before. Which was a fun experience and helped me learn a lot.

The main aspects of the game I worked on were, the character creator, and the collision on the hand-drawn map. I had wanted to make a character creator for a long time so it was enjoyable to finally get the opportunity and I think it turned out well.

We submitted the game on the itch.io jam and came 13th out of 30 entries. We were also mentioned in a list of favourites by one of the judges, found HERE.
