NOISE - JULY 2017

I had been wanting to make this game for a while after reading a website which had hundreds of wacky game ideas. The concept for the white noise only being noticable had been written out but I wanted to see what it actually looked like.

This was made in Gamemaker and didn't take very long, but it shows the concept of a pure noise game working well. There's a lot more you could do with this as essentially the concept is being blind except for motion.

You can use the arrow keys to move in the demo.
