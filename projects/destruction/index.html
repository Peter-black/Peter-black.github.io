Procedural Destruction (Honours Project) - Present (APRIL 2017)

For my Honours project I am creating a real-time procedural destruction system in Unreal Engine 4.

This system allows you to destroy any static mesh asset to an infinite degree (You can break the chunks as many times as you want). The destruction will form around the impact point and can be adjusted to simulate many materials such as glass, stone and wood.

The system uses a Voronoi based system to allow an artist to adjust the materials, with custom materials being easy to create. I have plans to develop the system into a plugin that can be sold on the Unreal Engine 4 marketplace.
