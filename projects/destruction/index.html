Procedural Destruction (Honours Project) - MAY 2017

For my Honours project I created a real-time procedural destruction system in Unreal Engine 4.

This system allows you to destroy any static mesh asset to an infinite* degree (You can break the chunks as many times as you want). The destruction will form around the impact point and can be adjusted to simulate many materials such as glass, stone and wood.

The system uses a Voronoi based system to allow an artist to adjust the materials, with custom materials being easy to create. However no artist pre-processing is required for my method, and it can be retroactively applied to any convex mesh.
