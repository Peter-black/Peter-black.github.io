WRENCH GAMES WEBSITE & BRANDING - AUGUST 2016

I was tasked with making the Wrench Games website and branding. This was something I had wanted to try for a while as I've always had an interest in graphic design and web development.

The wrench logo takes inspiration from the design of a classic NES/SNES controller and combines it with a clean and simple wrench shape. This allows the entire name to be derived from the logo without text.

The website uses bootstrap for the information sections and custom css for the project areas of the site.

I liked to put subtle animations on each of the project pages and it turned out well. I am particularly proud of the Habitat page which looks brilliant.
