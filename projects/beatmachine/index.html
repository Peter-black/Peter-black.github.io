Beat Machine - APRIL 2016

This is a simple Beat Machine made using PureData. It has 4 channels with 8 beats looping. You can choose to use a MIDI mode selecting MIDI notes per channel or you can load select Sample Mode and load in audio samples to play.

I had never created music before making this program so I found it very interesting to combine the logic of programming with musical concepts.

PureData was interesting to work with, the node based design was nice however I found it somewhat clunky to use. I would still consider using it for future audio based projects as its feature set is great.
