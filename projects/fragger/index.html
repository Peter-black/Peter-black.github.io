FRAGGER VITA - MAY 2015

This is a clone of the popular Miniclip flash game Fragger for the PlayStation Vita.

The aim of the game is to throw grenades to blow up all of the enemies.

This game uses the Box2D physics engine, and was the first game I made using the libraries.

To run the game you need to use a controller as there is no keyboard or mouse input.
