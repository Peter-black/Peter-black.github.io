SUPER CRATE BOX GBA - APRIL 2014

A simplified clone of the arcade game Super Crate Box made by Vlambeer. The game is about collecting as many crates as possible and avoiding enemies!

This is one of my favourite games , as it is a retro style game I decided to make a clone in a retro way, GBA style! This does mean that the game needs a GBA emulator to run. You can find the one I used here:
