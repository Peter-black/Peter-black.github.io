PATHFINDING TESTS - NOVEMBER 2013

I made two simple console-based pathfinding tests using different algorithms, written in C++.

The algorithms used are the Lee Algorithm and the A* Algorithm. The console outputs visually show the steps used to generate the final path and show the final number of steps it took for the program to reach the destination.
