TETRIX - MAY 2014

Tetrix is a flash game that combines two popular games, Tetris and Breakout. The game involves trying to score points in Tetris mode until you run out of time, then you clean up using a Breakout paddle and ball for more points!

This is a flash game written in ActionScript 3, I made this for a university project as I always thought it was an interesting game idea. I would like to make it into an app for mobile in the future.
