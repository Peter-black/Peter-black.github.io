kill spill (Global Game Jam 2017) - January 2017

The Global Game Jam 2017 had the theme 'Waves', which seemed too innocent. So we decided to make a game about battling waves of enemies and then finding yourself causing waves in a massive pool of their blood.

The main aspect we wanted to experiment with was the use of liquid simulation. We used Gamemaker and it has the Box2D LiquidFun plugin built in so we were able to simulate the liquids in 2D.

There were many setbacks along the way however in the end the game was well recieved by peers and fun to play overall. It fits our original vision reasonably well and sticks to the theme.
