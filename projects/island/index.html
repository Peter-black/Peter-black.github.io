THE ISLAND (SHADER DEMO) - NOVEMBER 2015

A small DirectX 11 and HLSL shader demo application. I used this application to learn and implement many different kinds of shaders, including techniquies such as tessellation, lighting and shadows.

This was my first DirectX application so it is just a simple scene with various shaders applied.

Shaders include:

  • Box Blur
  • Saturation
  • Tessellation
  • Shadow Maps
  • Multiple lights
  • Geometry Shaders
